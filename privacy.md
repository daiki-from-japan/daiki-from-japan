# Privacy Policy

**Last updated: May 1, 2026**

## 1. Introduction

This Privacy Policy describes how DAIKI (DAIKI | FROM JAPAN) handles information in connection with the personal automation tool used to upload videos to TikTok.

## 2. Information We Collect

This application is a personal single-creator automation tool. The following information is accessed:

- **TikTok OAuth access token**: Used solely to authenticate API requests to the creator's own TikTok account.
- **Video files**: Short-form video content created by the creator and uploaded to the creator's own TikTok account.

## 3. How We Use Information

The access token and video files are used exclusively to:
- Authenticate with TikTok's Content Posting API
- Upload video content to the creator's own TikTok account

No information is shared with third parties, sold, or used for any other purpose.

## 4. Data Storage

- Access tokens are stored locally on the creator's personal computer and are never transmitted to any server other than TikTok's official API endpoints.
- No viewer or follower data is collected or stored.

## 5. Third-Party Services

This application uses TikTok's API. TikTok's Privacy Policy applies to data processed by TikTok. Please review TikTok's Privacy Policy at: https://www.tiktok.com/legal/privacy-policy

## 6. Data Security

All data is handled locally on the creator's personal device. The application does not operate any server or database.

## 7. Children's Privacy

This application is not intended for use by minors and does not collect any data from children.
